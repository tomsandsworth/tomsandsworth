## 🛠 What I’m Working On (Closed Source)
*Most of my high-level work is proprietary or under NDA, but here’s what I build:*

* **[SpectralEngine](https://tomsandsworth.me)**: A custom, cross-platform 3D game engine built from scratch in **C++**. Uses a low-level rendering abstraction (`sokol_render.h`) to target Vulkan, WebGPU, and OpenGL.
* **InstantMVP for Unity**: A high-performance hot-reloading tool using **Roslyn** and **C# Reflection** to bypass Unity's domain reload cycle—turning minutes of waiting into seconds of coding.
* **Unity MCP Agent**: A Python/C# hybrid tool that gives LLMs direct access to the Unity Editor API for real-time scene manipulation and automated world-building.
